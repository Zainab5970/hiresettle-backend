# Changelog

All notable changes to this project will be documented in this file. See [standard-version](https://github.com/conventional-changelog/standard-version) for commit guidelines.

## 0.1.0 (2026-06-27)


### Features

* Add complete Swagger documentation for all endpoints, DTOs, and error responses
* Add /v1 global API prefix with health check and docs excluded
* Add Postman collection and environment files for API testing
* Add automated changelog generation and release tooling
* Add email/password authentication
* Add notification preferences feature
* Add user deactivation/reactivation functionality (admin only)
* Add retention timer endpoint for milestones
* Add user public profile endpoint
* Add on-chain event polling
* Add health check endpoint
* Add CRUD operations for engagements and milestones
* Add JWT-based authentication
* Add Stellar integration


### Bug Fixes

* Fix duplicate ResolveDisputeDto class
* Fix milestones controller req object not defined
